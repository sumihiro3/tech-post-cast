# Google Cloud Run環境構築とNestJSセットアップ

## 概要

有料機能のバックエンドとなるGoogle Cloud Run環境の構築

## 詳細

- Google Cloudプロジェクトのセットアップ
- NestJSアプリケーションのコンテナー化
- Cloud Runサービスの作成と設定
- 環境変数とシークレット管理の設定

- フロントエンドからAPIを呼び出せること
